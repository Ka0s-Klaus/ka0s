# `bin/`: utilidades y bootstrap

## Propósito
Herramientas auxiliares para tareas locales (setup, utilidades de CLI).

## Contenido típico
- Scripts de bootstrap (por ejemplo, setup de herramientas como `kubectl`).
- `README.md` con instrucciones de uso.

## Patrón Ka0s
- Scripts idempotentes y con salida clara.
- Cualquier uso operativo relevante debe tener documentación en `core/docs/`.
