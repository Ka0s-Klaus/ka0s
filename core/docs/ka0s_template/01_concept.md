# Ka0s Template - Concepto

## Descripción
Plantilla base para la creación de nuevos módulos en Ka0s. Define la estructura estándar que todos los directorios deben seguir.

## Estándar
*   Documentación modular (`01`, `02`, `03`).
*   Scripts en `scripts/` (si aplica).
*   Configuración en `config/` (si aplica).
