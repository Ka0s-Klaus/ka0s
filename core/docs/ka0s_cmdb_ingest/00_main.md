# Módulo: Ka0s CMDB Ingestion

Este módulo permite la gestión de la CMDB de iTop a través de archivos JSON en el repositorio, aplicando principios de Infraestructura como Código (IaC).

*   [Uso y Validación](02_usage_validation.md): Guía para crear CIs y realizar pruebas locales.
*   [Integración](03_integration.md): Detalles técnicos del script de ingesta y el workflow de GitHub Actions.
