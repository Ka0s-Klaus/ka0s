# Ka0s Workflow Statistics - Integración

## API de GitHub
Hace uso intensivo de la API v4 (GraphQL) y v3 (REST) de GitHub.
Requiere un token con permisos de lectura de metadatos (`KAOS_REPO_TOKEN`).

## Consumidores
Los datos generados por este módulo pueden ser consumidos por dashboards externos (ej. PowerBI, Grafana) o por el propio **Ka0s Dashboard** para visualizar la salud del proyecto.
