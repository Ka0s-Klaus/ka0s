# Respuesta del Agente Ka0s

**Pregunta:** ¿Cómo despliego un nuevo servicio?
**Fecha:** Fri Mar 13 19:50:57 UTC 2026
**Workflow ID:** 23067887168

---

Para desplegar un nuevo servicio en Ka0s, sigue estos pasos:

1. **Despliegue de repositorio Git**: Abre el repositorio Git de tu servicio en GitHub.
2. **Creación de un nuevo commit**: Haz un commit con los cambios necesarios para desplegar el servicio.
3. **Despliegue de servicios**: Ejecuta el comando `git push` para enviar los cambios al repositorio.
4. **Despliegue de servicios en Ka0s**: Abre el terminal y ejecuta el comando `cd-core-services.yml` para desplegar el servicio en Ka0s.
5. **Validación de resultados**: Verifica los logs del job `sync-cmdb` para asegurarte de que el servicio se haya desplgado correctamente.
6. **Validación de iTop**: Accede a la consola de iTop y verifica que el servicio se haya desplgado correctamente en la CMDB.

Por favor, ten en cuenta que estos pasos pueden variar dependiendo de las especificaciones de tu servicio y de las políticas de Ka0s. Si tienes alguna pregunta adicional, no dudes en preguntar.
