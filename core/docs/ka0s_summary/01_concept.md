# Ka0s Workflow Statistics - Concepto

## Descripción
Módulo de inteligencia de negocio (BI) aplicado a DevOps. Recopila métricas sobre el uso de GitHub Actions, eficiencia de los workflows y actividad del repositorio.

## Objetivos
*   Detectar workflows "flaky" (inestables).
*   Medir tiempos de ejecución para optimizar costes (minutos de runner).
*   Trazabilidad de Issues y Pull Requests.
