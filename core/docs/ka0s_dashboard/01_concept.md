# Concepto y Arquitectura: Ka0s Dashboard

## 1. Introducción
El Dashboard de Ka0s proporciona una interfaz visual para gestionar el estado del ciclo de vida del desarrollo y las operaciones, basado en metodologías ágiles (Kanban).

## 2. Arquitectura
*   **Frontend**: Generado estáticamente o mediante herramientas de visualización de GitHub Projects.
*   **Backend**: Basado en archivos JSON que representan el estado de las tareas (`backlog`, `in_progress`, `done`).
