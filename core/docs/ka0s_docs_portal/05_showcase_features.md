# Showcase de Nuevas Funcionalidades

Esta página demuestra las capacidades "vitaminadas" que hemos añadido a tu portal de documentación.

## 1. Pestañas de Contenido (Content Tabs)

Ideal para mostrar instrucciones diferentes según el sistema operativo o lenguaje.

=== "Linux / macOS"
    ```bash
    # Instalar dependencias
    pip install mkdocs-material
    ```

=== "Windows"
    ```powershell
    # Instalar dependencias
    pip install mkdocs-material
    ```

## 2. Teclas de Teclado (Keys)

Ahora puedes documentar atajos de teclado de forma elegante:

- Para copiar: ++ctrl+c++ o ++cmd+c++
- Para guardar: ++ctrl+s++
- Combinaciones complejas: ++ctrl+alt+delete++

## 3. Listas de Tareas (Tasklist)

Perfecto para checklists de procedimientos o seguimiento de progreso.

- [x] Configurar MkDocs
- [x] Añadir plugins avanzados
- [ ] Desplegar en Producción
- [ ] Celebrar el éxito :tada:

## 4. CriticMarkup (Revisiones)

Muestra cambios propuestos o correcciones directamente en el texto:

Esto es {--borrado--} y esto es {++añadido++}.
También puedes añadir {~~comentarios~>mejor usar esta palabra~~}.

## 5. Fecha de Actualización

Observa al pie de esta página (y de todas las demás). Verás la fecha de la última modificación del archivo en Git, gracias al plugin `git-revision-date-localized`.

## 6. Botones de Acción

- **Copiar Código**: Pasa el ratón por encima de cualquier bloque de código y verás un botón para copiar.
- **Editar Página**: Arriba a la derecha (junto a la tabla de contenidos) verás un icono de lápiz para editar esta página directamente en GitHub.
