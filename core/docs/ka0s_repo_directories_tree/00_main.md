# Ka0s: árbol completo de directorios

Este módulo publica un listado exhaustivo de directorios del repositorio para:
- Tener una visión de estructura.
- Facilitar preguntas tipo “¿qué hay en `<ruta>`?” sin perderse.
- Alimentar la batería de evaluación del agente.

- [Árbol de directorios](01_tree.md)
