# Mermaid Test Page

Esta página es para probar la integración de gráficos Mermaid.

## Gráfico de Flujo

```mermaid
graph TD
    A[Inicio] --> B{¿Funciona?}
    B -->|Sí| C[Genial]
    B -->|No| D[Revisar Config]
```

## Gráfico de Secuencia

```mermaid
sequenceDiagram
    Usuario->>Sistema: Petición
    Sistema-->>Usuario: Respuesta
```
