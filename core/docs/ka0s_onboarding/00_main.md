# Módulo Ka0s Onboarding

Guía técnica detallada para la incorporación de nuevos desarrolladores y colaboradores al ecosistema Ka0s. Este módulo centraliza todo el conocimiento necesario para configurar el entorno, entender los estándares y comenzar a aportar valor desde el primer día.

## Índice de Documentación

1.  [**Primeros Pasos y Requisitos**](./01_getting_started.md)
    *   Prerrequisitos del sistema (CLI, Lenguajes).
    *   Acceso a Repositorios y Credenciales.
    *   Estructura mental del proyecto.
2.  [**Entorno de Desarrollo (Local)**](./02_development_environment.md)
    *   Configuración del IDE (VSCode/Trae).
    *   Linters y herramientas de calidad local.
    *   Scripts de utilidad (`.github/scripts`).
3.  [**Flujo de Contribución**](./03_contribution_flow.md)
    *   Ciclo de vida de una tarea: Issue -> Rama -> PR.
    *   Convenciones de Commits y Versionado.
    *   Reglas de Oro (Normas Ka0s).
